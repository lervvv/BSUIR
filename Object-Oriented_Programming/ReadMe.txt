Программа реализует редактор фигур основываясь на основных принципах ООП: наследовании, инкапсуляции, абстракции и полиморфизме. Пользователь может создавать фигуры из предоставленных или подгружать плагины для добавления новых фигур в список. Реализована сериализация в файл в бинарном и текстовом виде и десериализация бинарных файлов. Пользователь при создании фигуры вводит размеры, координаты, цвет. После создания фигуры при нажатии на нее, появляется возможность ее изменения. 
Как работают основные функции:

Добавление фигур:
- пользователь выбирает тип фигуры
- заполняет параметры в специализированной форме
Фигура создается через фабричный метод (для плагинов) или напрямую

Редактирование фигур:
-клик на фигуру открывает форму редактирования
-клик на плагин - универсальная форма на основе рефлексии
-поддержка изменения параметров и цвета

Плагины:
-загружаются из DLL-файлов
-должны реализовывать IShapePlugin
-автоматически появляются в интерфейсе

Сохранение/загрузка:
-бинарная сериализация с учетом плагинов
текстовое сохранение в читаемом формате
-использование кастомного биндера для работы с плагинами
