Консольное приложение для сканирования локальной сети и обнаружения активных устройств. Программа определяет IP и MAC-адреса всех устройств в сети.

Основные функции:
-сканирование сети - автоматическое обнаружение активных сетевых интерфейсов
-Ping проверка - определение доступности устройств в сети
-MAC-адреса - получение физических адресов сетевых устройств
-асинхронное выполнение - параллельное сканирование для повышения скорости

Технические особенности
-ARP запросы - использование низкоуровневых сетевых функций для получения MAC-адресов
-фильтрация интерфейсов - работа только с активными сетевыми адаптерами
-расчет диапазона IP - автоматическое определение сети на основе маски подсети
