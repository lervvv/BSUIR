Данная программа представляет собой консольный клиент для взаимодействия с REST API файлового хранилища. Клиент предоставляет удобный интерфейс для выполнения различных операций с файлами и папками на сервере.

Основные функции
-управление файловой структурой
-просмотр дерева файлов: отображение всей файловой структуры сервера в виде древовидного списка
-создание папок: возможность создания новых директорий на сервере
-удаление папок: удаление директорий с подтверждением и возможностью рекурсивного удаления

Операции с файлами
-загрузка файлов: отправка локальных файлов на сервер
-чтение файлов: просмотр содержимого текстовых файлов непосредственно в консоли
-добавление текста: дописывание текста в конец существующего файла
-копирование файлов: создание копий файлов на сервере
-скачивание файлов: загрузка файлов с сервера на локальный компьютер
-удаление файлов: удаление файлов с сервера
-перемещение файлов: изменение расположения файлов на сервере

Архитектура и принципы
-REST API взаимодействие
-использование HTTP-методов (GET, POST, PUT, DELETE) для выполнения операций
-JSON-сериализация/десериализация для обмена данными
