Консольное приложение для взаимодействия с файловым хранилищем через GraphQL API. Предоставляет полный набор операций для управления файлами и папками.

Основные функции
-просмотр файлов - получение списка файлов и директорий
-чтение файлов - просмотр содержимого текстовых файлов
-управление файлами - удаление, копирование, перемещение
-редактирование - добавление текста и перезапись файлов

Технические особенности
-GraphQL API - использование GraphQL для эффективных запросов
-обработка ошибок - комплексная система обработки исключений
-асинхронность - неблокирующие операции
